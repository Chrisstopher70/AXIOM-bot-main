# AXIOM Memecoin Trading Bot

A professional, secure Telegram bot for trading memecoins on Solana with exceptional UI/UX.

## Features

- 🔐 **Secure Wallet Integration** - Connect via WalletConnect, Phantom, or Solflare
- 📈 **Real-time Trading** - Buy/sell memecoins with live prices
- 📊 **Portfolio Analytics** - Track P&L and performance
- 🔥 **Trending Coins** - Discover hot memecoins
- ⚡ **Fast Execution** - Lightning-fast transactions
- 🎯 **Professional UI** - Pyramid-shaped menu design

## Security Features

- ✅ Never asks for private keys or seed phrases
- ✅ All connections via secure wallet interfaces
- ✅ Encrypted user data
- ✅ Regular security audits
- ✅ Admin-only access to sensitive functions

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Copy `.env.example` to `.env` and configure
4. Run: `python main.py`

## Wallet Connection Methods

- **Phantom Wallet** - Browser extension
- **Solflare Wallet** - Mobile app
- **WalletConnect** - Universal standard
